## English

Dossier, a home for your files.

## Requirements

* Git
* Docker
* Docker Compose

## Installation

1. Clone the repository:

    ```sh
    git clone https://github.com/Felipe-Cavalca/dossier.git
    cd dossier
    ```

1. Build and start Docker containers:

    ```sh
    docker-compose up --build -d
    ```
